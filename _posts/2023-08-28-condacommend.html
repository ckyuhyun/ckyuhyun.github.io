This post introduces the basic commands on conda.

- Const activate is a command to create a new environment.

Conda create -n (Environment) python=(python version) 

    
        (base) > conda create -n virutal_env python=3.10

- Const list shows list of packages installed in the environment.

Conda list 

    
        (base) > conda list

- Const env list shows environments installed.

Conda env list 

    
        (base) > conda env list

- Const activate is to get an environment.

Conda activate (Environment) 

    
        (base) > conda activate tensorflow

- Remove an environment

Conda remove --name (Environment) --all 

    
        (base) > conda remove --name tensorflow --all

- Generate a requirements.txt

pip list --format=freeze > (file_name) 

    
        (base) > pip list --format=freeze > requirements.txt
