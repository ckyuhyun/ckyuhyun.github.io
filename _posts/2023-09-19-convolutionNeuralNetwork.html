The most object detections adopt the Convolutional Network Network(CNN) that is used to extract the features from input data. The main features of object detection are recognize and localize.

- Dense Convolutional networks (ResNet, VGG16) Image Reference

There are two anchors - positive(activated) or negatvie. The positive is for the objects mostly overlapped with the ground truth boxes whereas the negative is for mostly background almost less or never overlapped.

The network classify the object using binary cross-entropy loss.

- RPN(Region Proposal Network)
