This might be helping to setup a goold cloud API running in local machine.

Windows

Create a virutal environment. Reference

Const activate is a command to create a new environment.

1. Virtualenv Env is a module of Python Detail

2. Create a virtual environment 

    
        pip install virtualenv
        virtualenv <your-env>

2. Activate a virtual environment 

    
        <your-env>\Scripts\activate

3. Install Google Cloud Vision API if not installed yet. 

    
        <your-env>\Scripts\pip.exe install google-cloud-vision
